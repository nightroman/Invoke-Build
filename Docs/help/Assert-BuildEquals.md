# Assert-BuildEquals

```text
(equals) Verifies that two specified objects are equal.
```

## Syntax

```text
Assert-BuildEquals [[-A] Object] [[-B] Object]
```

## Description

```text
Scripts use its alias 'equals'. This command verifies that two specified
objects are equal using [Object]::Equals(). If objects are not equal the
command fails with a message showing object values and types.
```

## Parameters

```text
-A
    The first object.
    
    Required?                    false
    Position?                    0
```

```text
-B
    The second object.
    
    Required?                    false
    Position?                    1
```
